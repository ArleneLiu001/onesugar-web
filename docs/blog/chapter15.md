### 正则总结
#### 把小数0开头的多余0去掉：
```js
var a='00123';
alert(a.replace(/0*(\d+)/,"$1));
```

#### 留二位小数的正数
```js
/(^\d+)+(\.\d{0,2})?$/
```
#### 能输入大于0小于10亿的正数
```js
/(^\d{1,9})+(\.\d{0,2})?$/
```
#### 1000000内且限制保留八位小数的正数
```js
/(^\d{1,6})+(\.\d{0,8})?$/
```
#### 小于10亿的正数且限制保留六位小数的正数
```js
/(^\d{1,9})+(\.\d{0,6})?$/
```
#### 只能输入1-100之间的整数
```js
/^(100|[1-9][0-9]{0,1})$/
```
#### 0≤value<100”之间的数值
```js
/^(0|1|[1-9][0-9]{0,1})$/
```
#### 简单11位手机号码
```js
/^1\d{10}$/
```
